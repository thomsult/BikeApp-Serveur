# BikeApp-Serveur

Serveur backend pour l'application BikeApp.
Server développé en Laravel (PHP) pour gérer les requêtes et les données de l'application mobile.
Il offre une API RESTful pour permettre à l'application mobile de communiquer avec la base de données et de fournir une expérience utilisateur fluide.

WIP: Work in progress

## Technologies utilisées

- Laravel
- PHP
- MySQL
- JWT (JSON Web Tokens) pour l'authentification
- Eloquent ORM pour la gestion de la base de données
- RESTful API pour la communication avec l'application mobile
- Docker pour la conteneurisation de l'application
- Notifications push pour les mises à jour en temps réel
- Cookies pour la gestion des sessions utilisateur
- CORS pour permettre les requêtes cross-origin entre le serveur et l'application mobile
