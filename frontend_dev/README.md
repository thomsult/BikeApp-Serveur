# BikeApp-Application-PWA

Application frontend pour l'application BikeApp, développée en React et TypeScript, avec une architecture basée sur les composants fonctionnels et les hooks.

## Technologies utilisées

- VitePwa
- React
- TypeScript
- Mapbox
- TailwindCSS
- ShadCN
- Zustand
- Tanstack Query
- Tanstack Form
- Tanstack Router
- Zod
- Lucide
- Google Auth(OAuth2)
